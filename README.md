1. Upload a log file
2. The system parses:

   * Log levels (INFO, WARN, ERROR)
3. Performs analysis:

   * Error frequency
   * Pattern detection
   * Performance anomalies
4. Generates readable insights for developers

## ⚙️ Running Locally

### Prerequisites

* Node.js installed

### Setup

```bash
git clone https://github.com/your-username/smart-log-analyzer.git
cd smart-log-analyzer
npm install
```

### Environment Variables

Create `.env.local`:

```env
VITE_BASE44_APP_ID=your_app_id
VITE_BASE44_APP_BASE_URL=your_backend_url
```

### Run

```bash
npm run dev
```
